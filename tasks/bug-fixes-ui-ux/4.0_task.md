## Considerações de Localização

1. **Concordância de Gênero**
   - "Categoria" é feminino: "a categoria", "nova categoria"
   - "Tipo" é masculino: "o tipo", "novo tipo"

2. **Plural**
   - Categoria → Categorias
   - Receita → Receitas
   - Despesa → Despesas

3. **Capitalização**
   - Títulos: Primeira letra maiúscula ("Criar Categoria")
   - Botões: Primeira letra maiúscula
   - Sentenças: Primeira letra maiúscula

4. **Formalidade**
   - Usar "você" (informal, mas comum em apps)
   - Evitar "tu"
   - Mensagens diretas e claras

5. **Abreviações**
   - Evitar abreviações não-padrão
   - "ex:" é aceitável para "exemplo"

## =Ë Descrição

Esta tarefa estabelece a fundação técnica completa do projeto mobile, configurando o ambiente de desenvolvimento com React Native 0.82.0, Expo SDK 54.0.0 e TypeScript 5.3+. Inclui a estrutura de diretórios seguindo o padrão MVVM, configuração de ferramentas de build (EAS), linting, formatação de código e integração contínua (CI/CD). É a tarefa mais crítica do projeto, pois todas as demais dependem dela.

Este setup garante que todos os desenvolvedores tenham um ambiente consistente e produtivo desde o primeiro dia, com hot reload, type checking, e todas as ferramentas necessárias para desenvolvimento ágil. A configuração do EAS Build permite que builds nativos sejam gerados na nuvem, eliminando a necessidade de Xcode/Android Studio localmente para a maioria das tarefas de desenvolvimento.

O objetivo é ter uma aplicação "Hello World" funcional rodando em iOS e Android via Expo Go em menos de 3 horas após clonar o repositório.

## =Ý Subtarefas

### 1.1 - Inicialização do Projeto Expo
**Estimativa:** S
**Tipo:** Setup

**Descrição:**
Criar o projeto base usando Expo CLI com template TypeScript, configurar estrutura de diretórios MVVM e instalar dependências essenciais do MVP.

**Arquivos envolvidos:**
- `mobile/package.json`
- `mobile/app.json`
- `mobile/tsconfig.json`
- `mobile/babel.config.js`
- `mobile/metro.config.js`
- `mobile/src/App.tsx`

**Comandos/Código:**
```bash
# Criar projeto com template TypeScript
npx create-expo-app@latest mobile --template expo-template-blank-typescript

cd mobile

# Instalar dependências principais
npm install expo@~54.0.0 react-native@0.82.0 typescript@^5.3.0

# Instalar dependências de navegação
npm install @react-navigation/native@^6.0.0 @react-navigation/bottom-tabs@^6.0.0 @react-navigation/stack@^6.0.0
npm install react-native-screens react-native-safe-area-context

# State management e validação
npm install zustand@^4.5.0 zod@^3.22.0 react-hook-form@^7.48.0

# HTTP e storage
npm install axios@^1.6.0 @react-native-async-storage/async-storage@^1.21.0 expo-secure-store@~13.0.0

# Componentes e ícones
npm install lucide-react-native@^0.300.0 victory-native@^37.0.0

# Dev dependencies
npm install --save-dev @types/react@^18.2.0 @types/react-native@^0.72.0
```

**Checklist:**
- [ ] Projeto Expo inicializado com template TypeScript
- [ ] Todas as dependências instaladas sem erros
- [ ] Estrutura de diretórios MVVM criada
- [ ] App.tsx executando sem erros no Expo Go
- [ ] TypeScript configurado com strict mode
- [ ] Metro bundler configurando corretamente
